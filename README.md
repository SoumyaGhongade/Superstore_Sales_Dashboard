# Superstore_Sales_Dashboard (Tableau)

## 📊 Project Overview
This project presents a Tableau dashboard analyzing Superstore sales data.

## 🚀 Key Features
- Sales & Profit KPIs
- State-wise analysis
- Category insights
- Time-series trends

## 🛠️ Tools Used
- Tableau
- Excel

## 📷 Dashboard Preview
![Dashboard](dashboard.jpeg)

## 📈 Key Insights

- California and New York generate the highest sales among all states  
- Sales show fluctuations over time with noticeable growth trends  
- Categories like Phones and Chairs contribute significantly to revenue  
- Sales per customer indicates high-value repeat customers
